Principais formatações

Negrito / Destaque

Usando a tag b (não tem semâtica), a palavra fica em negrito.

Nesta frase, tenho um termo em destaque. usando a tag STRONG (semâtica).

Itálico / Ênfase

Usando a tag i (não é semâtica), o termo fica em itálico.

Nesta frase, temos um termo em ênfase usando a tag EM (semâtica).

Texto Marcado

Podemos criar um texto marcado usando uma tag MARK (style + background-color pra escolher a cor).

E no outro parágrafo, temos outro texto marcado no final.

Texto grande e pequeno

Texto Grande (tag big (está obsoleto)) e texto pequeno (tag small (não está obsoleto)).

Teto deletado

Podemos marcar um texto como como ixcluido (tag del) para indecar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido (tag ins) para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sumblinhado com a tag U (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20 + 3 (usar a tag sup).

Texto subscrito

Para inserir coisas do tipo H2O (usar a tag sub).
